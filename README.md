# Zentist Logos

Лёгкая статическая страница‑галерея для просмотра логотипов Zentist и связанных брендов. Логотипы собираются динамически из отдельных SVG‑слоёв, поддерживают переключение режимов отображения и тёмной темы.

## Возможности

- **Full Logo / Icon** — переключение между полным логотипом (иконка + название) и только иконкой.
- **Dark Mode** — переключение светлого и тёмного фона превью; названия автоматически инвертируются.
- **Адаптивная сетка** — 1/2/3 колонки в зависимости от ширины экрана.
- **Действия по наведению** — кнопки копирования/скачивания на карточке логотипа.
- Логотипы описываются декларативно в `script.js` и рендерятся из SVG‑слоёв в `assets/raw/`.

## Структура проекта

```
.
├── index.html       # Разметка страницы и контролы
├── styles.css       # Стили, темы и адаптивная сетка
├── script.js        # Конфигурация логотипов и рендеринг
└── assets/
    ├── raw/         # Исходные SVG-слои логотипов
    ├── icons/
    └── names/
```

## Запуск

Проект полностью статический — сборка не требуется. Достаточно открыть `index.html` в браузере.

Для корректной загрузки SVG лучше поднять локальный сервер:

```bash
# Python 3
python3 -m http.server 8000

# либо Node
npx serve .
```

Затем открыть [http://localhost:8000](http://localhost:8000).

## Добавление нового логотипа

1. Положите SVG‑слои в `assets/raw/`.
2. Добавьте объект в массив `LOGOS` в `script.js`, указав размеры (`fullW`/`fullH`), файл названия (`nameFile`) и слои иконки (`layers`).
